# Daily Standup Preparation

Automate your daily standup preparation to ensure you're always ready with meaningful updates.

## Actions

- [ ] Review yesterday's completed tasks in JIRA
- [ ] Identify today's priorities and goals
- [ ] Check for any blockers or impediments
- [ ] Update JIRA ticket statuses
- [ ] Prepare brief summary of accomplishments
- [ ] Note any help needed from team members
- [ ] Review team calendar for potential conflicts

## Reminders

- Remind me daily at 8:30am to prepare for standup
- Alert me 15 minutes before standup meeting
- Weekly reminder to review team sprint progress

## Communication Points

**What I accomplished yesterday:**
- [Auto-populate from work log]
- [Pull completed JIRA tickets]
- [Include documentation updates]

**What I plan to work on today:**
- [List prioritized tasks]
- [Include meeting commitments]
- [Note any dependencies]

**Any blockers or impediments:**
- [Check for blocked JIRA tickets]
- [Note waiting dependencies]
- [Identify resource needs]

## Integration Points

**JIRA Integration:**
- Pull completed tickets from yesterday
- Get assigned tickets for today
- Check for blocked or in-review items
- Update ticket statuses during standup

**Confluence Integration:**
- Reference relevant documentation updates
- Note any wiki pages created or updated
- Link to project documentation

**Slack Integration:**
- Post standup summary to team channel
- Share relevant links or updates
- Follow up on action items

## Automation Scripts

This workflow can trigger:
1. Morning JIRA ticket review
2. Calendar conflict detection
3. Automatic standup notes generation
4. Post-standup action item tracking
